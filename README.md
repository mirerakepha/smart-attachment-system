# Smart Attachment & Placement System
---
### This tool scraps popular websites to find internships/attachment opportunities for students and matches a student according to their skill, qualification and liking
---
#### Created using fastAPI and Vite
---

- How to get started

 Clone the reposittory

```
 git clone https://github.com/mirerakepha/smart-attachment-system.git
```

```
cd backend 
```

```
pip install -r requirements.txt
```
- Test the backend

```
uvicorn app.main:app --reload
```


```
cd ..
```

- Run the frontend
```
npm run dev
```
